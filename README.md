# 项目启动指南

## 前置条件

- 确保已安装 **Node.js**（推荐 v18 或更高版本）
- 确保已安装 **pnpm**（若未安装，执行 `npm install -g pnpm`）

---

## 快速开始

### 1. 构建数据结构层

在项目根目录下，进入数据层包并执行构建命令：

````bash
cd packages/datalayer
pnpm run build


### 2. 安装依赖并启动应用

进入应用目录，安装依赖后启动开发服务器：

```bash
cd apps/app
pnpm install
pnpm run dev

### 3. 访问应用

开发服务器启动成功后，在浏览器中打开：

```bash
http://localhost:5173

````
